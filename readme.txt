Adds the ability to create easily notification in the admin panel

== Description ==

This plugin helps admins to easily create admin notification inside the WordPress administration.

You can choose many options and decide how and where to display the notifications.

== Installation ==

1. Upload the entire 'easy-admin-notification' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Look at your admin bar under 'Appearance' menu
4. Start displaying notification and smile...
